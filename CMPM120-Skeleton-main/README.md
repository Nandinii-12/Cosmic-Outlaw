# Cosmic Outlaw 

# Controls 
- Use "a" to move left and "d" to move right
- Click the spacebar to shoot the gun

# Scoring
- +1 points for every alien you shoot down 
- You don't get a point if the green alien leaves the screen
- -1 heart if you get hit by a laser shot by an alien
- -2 hearts if you get hit by a green alien

# Theme
- Space cowboys!

# Bonus elements
- Multiple distinct levels: I have 4 clear levels 
    - Number of aliens increase every level
- Title screen
    - Displays game name
    - Shows controls
